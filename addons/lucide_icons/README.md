# Lucide Icons for Godot 4

Local Lucide-style icon package.

Paths used by this app:
- `res://addons/lucide_icons/icons/file-plus-2.svg`
- `res://addons/lucide_icons/icons/user-round.svg`
- `res://addons/lucide_icons/icons/arrow-right.svg`
- `res://addons/lucide_icons/icons/menu.svg`

You can also enable the plugin in **Project > Project Settings > Plugins** and add a `Lucide` node from the Create Node dialog.
